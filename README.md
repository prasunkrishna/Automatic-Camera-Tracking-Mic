# Automatic Camera Tracking using Microphone Localization

This project implements an **automatic camera tracking system** using ESP32-S3 and ESP8266 with WiFi FTM distance estimation and microphone audio streaming.

## Components
- ESP32-S3 (I2S microphone + FTM ranging)
- ESP8266 anchor node
- Python processing scripts
- Servo Motors

## Features
- WiFi FTM based distance localization
- I2S microphone streaming over UDP
- Speaker direction detection
- Camera tracking control
